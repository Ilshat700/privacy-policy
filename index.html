Политика конфиденциальности для приложения "Калькулятор ГИИ"

Последнее обновление: 14.11.2024

1. Введение

Данное приложение "Калькулятор" (далее - "приложение") разработано для выполнения математических вычислений. Мы ценим вашу конфиденциальность и соблюдаем правила защиты данных.

2. Сбор данных

Приложение не собирает, не хранит и не обрабатывает личные данные пользователя. Оно также не требует доступа к таким функциям устройства, как контакты, местоположение или файлы пользователя.

3. Безопасность данных

Так как приложение не собирает данные, дополнительные меры безопасности для защиты данных не требуются. Приложение работает локально на устройстве пользователя и не подключается к каким-либо сторонним сервисам для передачи информации.

4. Сторонние сервисы

Приложение не использует сторонние сервисы и библиотеки для обработки данных, а также не передаёт данные третьим лицам.

5. Изменения политики

Мы оставляем за собой право вносить изменения в данную политику конфиденциальности. При изменении содержания политики пользователи будут уведомлены через обновлённое описание приложения.
